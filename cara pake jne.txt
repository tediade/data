Butuh Modul Pengiriman JNE gratis untuk Toko Online Opencart Anda? Humayraa dan bestariweb akan berbagi modul ini secara gratis.

Cara Install
Download modul disini
Upload ke root folder opencart kemudian extract.
Login ke halaman admin
Klik Extension , Shipping
Klik icon + di baris JNE Shipping, kolom Action, untuk install module
Jika Proses Install berhasil, tanda + berubah jadi tanda minus
Menambah Zone JNE
Klik System, Localisation, Geo Zones
Klik icon + di kanan atas layar untuk menambahkan Geo Zone JNE
Isi Nama Geo Zone : JNE Regular
Isi Description : JNE Regular
Klik icon + dibawah kolom Zone untuk menambahkan Geo Zone ke Zone
isi Country: Indonesia
Isi Zone: All Zone
Klik icon Disk (save) di kanan atas layar untuk mentimpan geo zones
Setting JNE Rate
Klik Extension , Shipping
Klik icon pensil di baris JNE Shipping, kolom Action, untuk edit module
Isi Untuk General Setting
Isi Tax Class: None
Isi Status: Enable
Isi Sort Order: 0
Klik Tab JNE Regular
Isi Rate dengan isi file Tarif JNE Reguler JKT yg ada di module ini
Format isi file = kodepos:tarif, jika ada perubahan, silahkan sesuaikan
Isi Status: Enable
Klik Save (kanan atas Layar)
